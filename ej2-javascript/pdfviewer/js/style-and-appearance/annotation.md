# Style and Appearance in Annotation module ##Platform_Name## Pdfviewer control

Use the following CSS to customize the Highlight icon.

```

  .e-pdfviewer .e-pv-highlight-icon{
    color: rgb(143, 89, 219);
    font-size: 20px;
  }
```
