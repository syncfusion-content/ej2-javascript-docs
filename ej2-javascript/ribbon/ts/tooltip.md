# Tooltip

The Ribbon control supports tooltips to show additional information on Ribbon items. The tooltip appears when the user hovers over a Ribbon item.

## Adding Title

The [title](https://ej2.syncfusion.com/documentation/api/ribbon/ribbonTooltip/#title) property can be used to set the tooltip title for each Ribbon item.
