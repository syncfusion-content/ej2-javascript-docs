# ErrorBar

`ErrorBar` module is used to render the error bar for series.

## Methods

### doErrorBarAnimation

Animates the series.

| Parameter | Type | Description |
|------|------|-------------|
| series |  [`Series`](./api-series.html) | Defines the series to animate. |

Returns *void*

### render

Render the error bar for series.

Returns *void*
