# MajorTickLines

Configures the major tick lines.

## Properties

### color `string`

The color of the major tick line that accepts value in hex and rgba as a valid CSS color string.

Defaults to *null*

### height `number`

The height of the ticks in pixels.

Defaults to *5*

### width `number`

The width of the tick lines in pixels.

Defaults to *1*
