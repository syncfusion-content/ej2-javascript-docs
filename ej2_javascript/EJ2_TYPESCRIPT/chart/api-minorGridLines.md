# MinorGridLines

Configures the minor grid lines in the `axis`.

## Properties

### color `string`

The color of the minor grid line that accepts value in hex and rgba as a valid CSS color string.

Defaults to *null*

### dashArray `string`

The dash array of grid lines.

Defaults to *''*

### width `number`

The width of the line in pixels.

Defaults to *0.7*
